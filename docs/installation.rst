============
Installation
============


Stable release
--------------

To install JSON Clever Key Sorter, run this command in your terminal:

.. code-block:: console

    $ pip install json_clever_key_sorter

This is the preferred method to install JSON Clever Key Sorter, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for JSON Clever Key Sorter can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/rbanffy/json_clever_key_sorter

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/rbanffy/json_clever_key_sorter/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/rbanffy/json_clever_key_sorter
.. _tarball: https://github.com/rbanffy/json_clever_key_sorter/tarball/master
